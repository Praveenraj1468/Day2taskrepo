
1.Difference between Document vs Window Object:-

  Document Object:-

        The Document Object is your html, aspx, php, or other document that will
be loaded into the browser.

        It auctually gets loaded inside the Window Object and has properties ava-
-ilable to it like title, URL, cookies, etc.

        If you want to access a property for the window it is window.property, if
it is document it is window.document.property which is also available in short as
document.property.

  
  Window Object:-

        The Window Object is the first thing that gets loaded into browser and it
is the main Java Script object root in a browser,also can be traeted as the root 
of the DOM.

        It has the majority of the properties like length, innerWidth, innerHeight, 
name, if it has been closed, its parents and more.

        It is the first thing to load into the browser and it is called as global
object.
